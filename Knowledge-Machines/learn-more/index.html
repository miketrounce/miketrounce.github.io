Knowledge Machines

Learn More

Knowledge Machines are dashboard websites. The technology behind their appearance is Vite. They allow you to monitor a specific data-driven situation, such as shipping activity in the Taiwan Strait.

They come with a database (typically Supabase) which contains the data you see on the dashboard. You can fetch that data via API.

The method used to get data for the database is a scheduler — work is done regularly to go and fetch ("pull") data from somewhere, e.g. X or Instagram. This is the data pipeline, and nowadays this is managed by an agent such as Claude Code Scheduler rather than a script and cron job.

As an audit trail, screenshots of social media posts are taken and stored in an object storage database. It is from these that data is extracted and ingested.

Knowledge Machines are run in the cloud. The code is stored on GitHub, and the website is hosted for free using GitHub Pages.
